# Как узнать ID группы/супергруппы в Telegram

## Способ 1: Через бота @userinfobot

1. Добавьте бота [@userinfobot](https://t.me/userinfobot) в вашу группу
2. Отправьте любое сообщение в группу
3. Бот ответит с информацией о группе, включая ID
4. ID группы будет отрицательным числом (например: `-1001234567890`)

## Способ 2: Через веб-версию Telegram

1. Откройте группу в веб-версии Telegram: https://web.telegram.org
2. Посмотрите URL в адресной строке
3. ID группы будет в URL после `/c/` (например: `https://web.telegram.org/k/#-1001234567890`)

## Способ 3: Через вашего бота (временно)

Добавьте в код бота временную команду для получения ID:

```python
async def get_chat_id(update: Update, context: ContextTypes.DEFAULT_TYPE):
    """Временная команда для получения ID чата"""
    chat_id = update.message.chat.id
    await update.message.reply_text(f"ID этого чата: {chat_id}")

# В main() добавьте:
application.add_handler(CommandHandler("getid", get_chat_id))
```

Затем отправьте `/getid` в группе, и бот ответит с ID.

## Важно

- ID обычной группы начинается с `-` (например: `-123456789`)
- ID супергруппы начинается с `-100` (например: `-1001234567890`)
- Используйте полный ID, включая знак минуса

## Как узнать ID топика

Если группа использует топики, и вы хотите, чтобы бот отправлял ответы в конкретный топик:

1. Отправьте команду `/getid` в нужном топике группы
2. Бот ответит с ID топика (положительное число, например `12345`)
3. Используйте этот ID для переменной `TELEGRAM_TOPIC_ID`

**Важно:** ID топика - это положительное число, в отличие от ID группы (который отрицательный).

## Пример настройки

```powershell
# Windows PowerShell
$env:TELEGRAM_GROUP_ID="-1001234567890"
$env:TELEGRAM_TOPIC_ID="12345"  # Опционально: ID топика для ответов
```

```bash
# Linux/Mac
export TELEGRAM_GROUP_ID="-1001234567890"
export TELEGRAM_TOPIC_ID="12345"  # Опционально: ID топика для ответов
```
